# 🧠 System Design Interview Checklist

Before solving any problem always discuss:

- Functional Requirements
- Non Functional Requirements
- Scale Estimation
- Database Design
- API Design
- High Level Architecture
- Deep Dive Components
- Bottlenecks
- Scaling Strategy
- Trade Offs
- Security
- Monitoring

---

# 🎯 Best Practice Strategy

For every design question:

1. Clarify requirements
2. Estimate scale
3. Design APIs
4. Create high level architecture
5. Deep dive into core components
6. Discuss database choices
7. Explain scaling strategy
8. Explain bottlenecks and solutions
9. Discuss trade offs
10. Conclude confidently
